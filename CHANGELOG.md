# Changelog

All notable changes to UnionStacks will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive documentation with SwiftDocC
- How-to guide for building wrapping HStacks
- Advanced patterns and examples
- GitHub Pages deployment workflow

## [1.0.0] - 2024-12-23

### Added
- Initial release of UnionStacks
- `WrappingHStack` - Auto-wrapping horizontal flow layout
- `CStack` - Centered layout with balanced distribution
- Customizable horizontal and vertical spacing
- Full SwiftUI Layout protocol implementation
- iOS 17+, macOS 14+, tvOS 17+, watchOS 10+ support
- Swift 6 language mode support
- Complete inline documentation
- Swift Package Manager support

### Features
- Automatic content wrapping for dynamic collections
- Perfect centering for navigation bar layouts
- Efficient O(n) layout algorithm
- Responsive design that adapts to screen sizes
- Support for variable-width content

## [0.1.0] - 2024-12-22

### Added
- Initial development version
- Basic `WrappingHStack` implementation
- Basic `CStack` implementation
